# Bac IA - Tutorat IA Premium pour le Maroc 🇲🇦

Bac IA est une plateforme de tutorat IA personnalisée pour les lycéens marocains préparant le Baccalauréat. 

## 🚀 Fonctionnalités
- **Tuteur IA Expert** : Connaît parfaitement le programme officiel marocain (SM, PC, SVT, Éco, etc.).
- **Interface Premium** : Design minimaliste avec glassmorphism, optimisé pour le confort visuel (Dark Mode).
- **Suivi Parent** : Dashboard dédié pour suivre la progression des enfants.
- **Support Multilingue** : Français, Arabe et Darija.

## 📂 Structure du Projet
- `designs/` : Maquettes HTML/CSS haute fidélité générées via Stitch.
- `.stitch/` : Configuration du design system et des skills d'IA.

## 🎨 Design System
Le projet utilise le système **Premium Dark Glass** :
- **Accent** : #0F766E (Teal)
- **Background** : #0A0A0B (Deep Dark)
- **Styles** : Glassmorphism (blur 20px), Roundness 12px.
